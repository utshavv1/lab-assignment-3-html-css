# LAB ASSIGNMENT 3
## Designing a Static Management System Web Interface Using HTML and CSS

### Description
This project is a static Student Management System interface created using HTML5 and CSS3.
The design follows normal document flow without using Flexbox, Grid, Position, or Media Queries.

### Pages Included
- Home Page
- Login Page
- Dashboard Page
- Student List Page

### Technologies Used
- HTML5
- CSS3

### Folder Structure
- index.html
- login.html
- dashboard.html
- list.html
- style.css
- img.jpg
